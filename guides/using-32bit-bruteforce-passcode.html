Using 32bit-Bruteforce-Passcode

Overview

Bruteforcing a passcode (using 32bit-bruteforce-passcode) is a way of getting the passcode on 32-bit iOS devices, so you can unlock the device.

Prerequisites

What You'll Need

A computer running Linux or macOS

Step-by-Step Guide

Step 1: Setting Up

Download the utility from the link provided, and extract it.

https://github.com/tuanemss/32bit-Bruteforce-Passcode/releases/tag/new/

Put the device into DFU mode, then follow step 2.

Step 2: Using the tool

Open up terminal, and navigate to where you extracted the .zip file.

Then, run this command in terminal.

  • ./bruteforce.sh

From there, it should continue the bruteforcing automatically. Your passcode should be displayed on screen after some time.

Step 3: Reboot

Reboot your device to apply the changes by either holding the power and home buttons for 10 seconds. Your device should now boot into the lock screen, where you can enter the passcode.
